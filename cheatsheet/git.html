a brief summury:

.git is version control system with three steps:

  1. working directory: here is being played. the file is added
  2. index: here has being seen the changes/results. ready to committed
  3. repository: is the place where the changes are kept. has been committed

basic commands

  • git init
  • git status
  • git add dir.ext
  • git commit -m "a text message"
  • git log
  • git rm old-name new-name
  • git rm file-name
